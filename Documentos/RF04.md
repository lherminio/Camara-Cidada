# Consulta e Busca Pública
**COMO** Cidadão/Eleitor,  
**QUERO** pesquisar por palavras-chave ou autores  
**PARA** encontrar leis de meu interesse e fiscalizar o trabalho dos parlamentares.

**Critérios de Aceitação:**
- Deve existir um campo de busca textual que pesquise no título e na ementa dos projetos.
- O sistema deve permitir filtrar os resultados por parlamentar (autor) ou por categoria.
- A lista de resultados deve exibir o status atual de cada projeto (Ex: "Virou Lei" ou "Arquivado").
