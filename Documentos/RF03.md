# Registro de Votações Nominais
**COMO** Operador do Sistema,  
**QUERO** lançar o resultado das votações de cada projeto  
**PARA** registrar como cada parlamentar se posicionou oficialmente.

**Critérios de Aceitação:**
- O sistema deve listar todos os parlamentares ativos para a marcação do voto.
- As opções de voto devem ser restritas a: Sim, Não, Abstenção ou Ausência.
- O sistema deve calcular automaticamente o resultado final (Aprovado/Reprovado) com base nos votos inseridos.
- Uma votação só pode ser registrada se estiver vinculada a um projeto já cadastrado.
