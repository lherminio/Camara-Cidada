# Cadastro de Parlamentares
**COMO** Operador do Sistema ,  
**QUERO** cadastrar os vereadores e seus perfis oficiais  
**PARA** que a população saiba quem compõe a legislatura atual e quais seus partidos.

**Critérios de Aceitação:**
- Deve ser possível informar nome completo, nome parlamentar, partido e biografia.
- O sistema deve permitir o upload de uma foto oficial.
- O sistema deve impedir o cadastro de parlamentares com o mesmo nome parlamentar na mesma legislatura.
- Deve ser possível editar ou inativar um parlamentar (ex: em caso de licença ou renúncia).
