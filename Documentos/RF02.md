# Registro de Proposições (Projetos e Leis)
**COMO** Operador do Sistema,  
**QUERO** cadastrar os projetos de lei, indicações e requerimentos  
**PARA** manter o histórico de produção legislativa organizado e consultável.  

**Critérios de Aceitação:**
- Deve ser possível informar título, número do projeto, ano, autor e uma breve ementa (resumo).
- O sistema deve permitir a classificação por categoria (Ex: Saúde, Educação, Infraestrutura).
- O sistema deve exigir um status inicial (Ex: Em Tramitação).
- Deve ser possível anexar o link ou o arquivo PDF do projeto original.
